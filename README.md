# SHENNONG-ISSP

SHENNONG-ISSP 是一款用于**茶叶样品鉴别**的 Windows 桌面端软件。

> 当前仅发布 Windows 安装包（.exe），源码暂不公开。

## 下载
请前往本仓库的 **Releases** 下载最新版本安装包：`Shennong-ISSP-Setup.exe`。

## 安装与运行
1. 下载 `Shennong-ISSP-Setup.exe`
2. 双击安装，按提示完成
3. 从开始菜单或桌面快捷方式启动软件

## 系统要求
- Windows 10/11（64-bit）

## 常见问题
- 若 Windows 提示“已保护你的电脑/未知发布者”：点击“更多信息”→“仍要运行”。
